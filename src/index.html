Recipe App

Description:

A movie/tv show lookup app built with HTML, SCSS & Vanilla JavaScript that allows an user to lookup information on tv shows and movies. An user can use the search field to search for a specific movie/tv show or they can select a movie/tv show that's "trending" or a genre specific movie/tv show. Additional features of this app include a dark/light mode that saves the users preference to local storage. The data for this application was fetched from TheMovieDB API.
