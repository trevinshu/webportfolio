Projects:

Movie/TV App

Description:

I built this movie/tv show web application with HTML, SCSS & Vanilla JavaScript which allows a user to lookup information on tv shows & movies. In order to do this a user can use the search input field to search for a specific movie/tv show they are looking for or they can select a movie/tv show from the pre-made categories that are displayed on the homepage when the page is loaded. In order to get and display this data I used TheMovieDB API.

Additional features include a dark/light mode which saves the users color mode preference to local storage so the next time they use the site the site UI will match their preferred color scheme.

Weather App

Description:

I built this weather web application with HTML, SCSS & Vanilla JavaScript which allows a user to lookup the weather of a specific location. In order to search the user has to enter the city & country in the appropriate input fields and hit search. The banner image will change based on the location the inputted so if a user searches Tokyo the banner image will change to something related to Tokyo. Other features of this application are a toggle to switch from Celsius to Fahrenheit and vise versa.
