Projects:

Movie/TV App

Description:

Using TheMovieDB API I built this fully responsive web application with HTML, SCSS & Vanilla JavaScript that allows an user to lookup information on tv shows & movies. In order to find a movie a user can use the search input field to search for a specific movie/tv show they are looking for or they can select a movie/tv show from a list of pre-made genre categories that are displayed on the homepage when the page is loaded. I also custom built an arrow slider that swaps through pages of movies & tv shows.

Additional features include a dark/light mode which saves the users color mode preference to local storage so the next time they use the site the site UI will match their preferred color scheme.

Recipe App

Description:

Using TheMealDB API I built this fully responsive web application with HTML, SCSS & Vanilla JavaScript that let's an user lookup recipes. The user has two options to find recipes. They can select a food category from the list of food categories that are displayed on page load which will then display a number of recipes that match that category or are closely related to that category. They can also search for a recipe which will output a list of recipes that match or are closely related to the search term provided by the user. I also custom designed & implemented a modal that shows a more detailed version of the recipe that the user clicked on.

Weather App

Description:

Using the OpenWeather API I built this fully responsive web application with HTML, SCSS & Vanilla JavaScript which allows an user to lookup the weather of a specific location. In order to search the weather the user has to enter both the city & country code of the location they need the weather from in the appropriate input fields and hit search. When a search is performed the banner image will swap to an image that's relevant to the location they searched for & this was accomplished using Unsplash. Finally I included a toggle that when clicked switches from Celsius to Fahrenheit & vise versa without reloading the page.
